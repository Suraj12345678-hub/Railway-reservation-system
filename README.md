🚂 Railway Reservation System
A comprehensive web-based application designed to automate and streamline the process of train ticket booking. This project focuses on managing train schedules, seat availability, and user bookings efficiently.

🚀 Key Features
User Authentication: Secure signup and login system for passengers.

Train Search: Facility to search for trains based on source, destination, and date.

Real-time Booking: Instant ticket booking with seat allocation logic.

Admin Dashboard: Tools for administrators to add/update trains, manage schedules, and view all bookings.

Booking History: Users can view their past and upcoming journey details.

Ticket Cancellation: Option for users to cancel their booked tickets.

🛠️ Tech Stack
Frontend: HTML5, CSS3, JavaScript, Bootstrap

Backend: PHP

Database: MySQL

Server: XAMPP / Apache

⚙️ How to run the Railway reservation system using PHP and MySQL

1. Download the  zip file

2. Extract the file and copy Railway folder

3.Paste inside root directory(for xampp xampp/htdocs, for wamp wamp/www, for lamp var/www/html)

4. Open PHPMyAdmin (http://localhost/phpmyadmin)

5. Create a database with name rail_db

6. Import Railway file(given inside the zip package in SQL file folder)

7.Run the script http://localhost/Railway (frontend)

8. Link for admin Panel : http://localhost/Railway/admin
